## Come usare
1. Compila i segnaposto `{{…}}` nella **Lettera** e nella **Bozza DPA** con i dati dell'Istituto.
2. Invia: Lettera + `Pacchetto-DPO-pantedu.pdf` (Allegato A) + `Bozza-DPA-Art28.pdf` (Allegato B).
3. Su richiesta del DPO: fornisci l'Allegato C (report firmato) e la DPIA.
4. Firma il DPA quando concordato (firma digitale PAdES raccomandata).

> Nota di trasparenza: la documentazione attesta *due diligence* (Art. 24/32); un pentest manuale certificato di terza parte resta opzione a richiesta del DPO, a spese della scuola.
