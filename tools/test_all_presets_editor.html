Test tutti i preset liste — Pantedu

Come usarlo:
  1. Apri l'editor inline di un quesito su Pantedu
  2. Posizionati nel campo QUESITO (svuotalo se serve)
  3. Clicca "Inietta nell'editor" qui sotto per copiare l'HTML
  4. Incolla con Ctrl+V nel campo Quesito
  5. Salva → verifica preview + .fm-collection post-save + ricompila PDF
Cosa testa:
  • Tutti i 9 preset ordered (numerici/alfa/roman/zero + suffisso . o ))
  • 2 preset unordered (arrow-bullet, star-circle) + default disc
  • 3 livelli annidati per ogni preset
  • Inline format (bold, italic, underline) dentro <li>
🚀 Iniezione completa one-shot (Console DevTools):

Apri DevTools (F12) sull'editor, incolla nella Console → tutti i preset vengono iniettati nel campo QUESITO attivo.
